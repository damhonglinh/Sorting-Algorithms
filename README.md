Sorting-Algorithms
==================

Implementation of 6 sorting algorithms:
<ul>
  <li>Insertion Sort</li>
  <li>Merge Sort</li>
  <li>Heap Sort</li>
  <li>Quicksort with Hoare partition</li>
  <li>Quicksort with Median partition</li>
  <li>Quicksort with Random partition</li>
</ul>
